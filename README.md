# SysInit New
### SysInit. Refined version as I couldn't get the first to work well.
A Linux environment setup automation script that takes care of XDG Specifications, tool home
environment variables, and installing Oh My Zsh as well as Zsh and making Zsh default.

### Usage
Clone this Github repository:
```
git clone https://github.com/ShadowPlasma2009/SysInit.git
cd SysInit
```

Configure execution permissions and run `main.sh`:
```
chmod +x main.sh
./main.sh
```

Log out and log back in for the changes to take place.

Footnote: Unlike the previous script, this one isn't destructive and won't destroy your home directory. I am not responsible for any unexpected behaviour. No AI was used except a few times I let Copilot generate commit notes.
